UbuntuのClibor的存在CopyQ

 2019-6-30 |  2020-5-14 |  1 min read
Ubuntu CopyQ

概要

普段 Windows では ClipBoard の履歴にcliborを使っています。
最近 OS を ubuntu で利用することがあり、ubuntu でコピー履歴管理を探していたところ、
Windows/MacOS/Linux 対応の CopyQってのが見つかりました。

さわってみて

Clibor みたいに、ショートカットにAlt*2とかの設定はできませんでしたが、
無変換に割り当てられたので概ね満足。

CopyQ には、履歴管理以外にも機能があるみたいですが、
とりあえずやりたいことができたので OK。
