UbuntuのClibor的存在CopyQ

 2019-6-30 |  2020-2-19 |  1 min read

概要

普段WindowsではClipBoardの履歴にcliborを使っています。
最近OSをubuntuで利用することがあり、ubuntuでコピー履歴管理を探していたところ、
Windows/MacOS/Linux対応のCopyQってのが見つかりました。

さわってみて

Cliborみたいに、ショートカットにAlt*2とかの設定はできませんでしたが、
無変換に割り当てられたので概ね満足。

CopyQには、履歴管理以外にも機能があるみたいですが、
とりあえずやりたいことができたのでOK。
