zsh上でhistoryコマンドを打つと直近15件しかでてこない

 2020-5-30 |  2020-5-30 |  1 min read
zsh shell command history

概要

bash を使ってたときは、historyで全履歴が表示されたのに対して、 zsh ではhistoryを叩いても、直近 15 件しか表示されなくて困ってました。

原因

bash のhistoryと zsh のhistoryコマンドでは仕様が違う。 両方とも fc コマンドの alias として用意されているみたいですけど、alias のはられ方が違うのかも。詳細は調べていないので、知っている方いたら教えてください。

デフォを全件表示にする

1

alias history='history 1'

history 1で 1~現在までの履歴が表示されるので、historyに alias を貼る。 まぁ何が困ってたかって、grep 検索できないことだったんで、zsh の検索用 plugin 使ったほうが良いのかもね。
